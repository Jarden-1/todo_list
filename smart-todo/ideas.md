# SmartTodo 设计方案

## 三种设计方向

### 方向 A：极简工程风（Minimal Engineering）
冷静克制的工具美学，以代码编辑器为灵感，深色背景、等宽字体点缀、精确的间距系统。
**概率：0.05**

### 方向 B：现代纸质感（Modern Paper）
以纸张、卡片为隐喻，米白底色、柔和阴影、手写感衬线字体，营造专注书写的氛围。
**概率：0.07**

### 方向 C：精准深色仪表盘（Precision Dark Dashboard）✅ 选定
深色背景配合精心设计的信息层次，蓝紫渐变作为品牌色，玻璃态卡片、微妙的光晕效果，兼具专业感与现代感。
**概率：0.03**

## 选定方向：精准深色仪表盘（Precision Dark Dashboard）

### Design Movement
Glassmorphism + Precision UI —— 以深色为基底，玻璃态卡片浮于层次之上，精确的信息密度，专业工具的质感。

### Core Principles
1. **信息密度优先**：每个像素都有意义，避免装饰性空白，但保留呼吸感
2. **层次清晰**：背景 → 面板 → 卡片 → 交互元素，四层深度
3. **色彩克制**：深色底 + 一个品牌色（蓝紫渐变），优先级/状态用色彩编码
4. **动效服务功能**：过渡动画传递状态变化，不炫技

### Color Philosophy
- 背景：`#0F0F1A`（深蓝黑）
- 面板：`#161625`（略亮的深蓝）
- 卡片：`rgba(255,255,255,0.04)` 玻璃态
- 品牌色：`#6366F1`（靛蓝）→ `#8B5CF6`（紫罗兰）渐变
- 文字主色：`#F1F5F9`
- 文字次色：`#94A3B8`
- 成功绿：`#10B981`
- 警告橙：`#F59E0B`
- 危险红：`#EF4444`

### Layout Paradigm
三栏式：固定左侧导航（240px）+ 主内容区（自适应）+ 右侧详情面板（380px，按需展开）
添加区固定在主内容区顶部，不随列表滚动

### Signature Elements
1. **渐变品牌色**：按钮、激活态、进度条使用蓝→紫渐变
2. **玻璃态卡片**：`backdrop-blur` + 半透明边框 + 微妙内阴影
3. **优先级色带**：卡片左侧 3px 竖线，颜色编码优先级

### Interaction Philosophy
- 悬停时卡片轻微上浮（translateY -2px）
- 点击有 scale(0.98) 反馈
- 面板滑入滑出，不突兀跳变
- AI 整理时有加载动画，完成后有成功反馈

### Animation
- 卡片进入：`opacity 0→1` + `translateY 8px→0`，stagger 40ms
- 面板展开：`translateX 100%→0`，200ms ease-out
- 模态框：`scale(0.96)→1` + `opacity 0→1`，180ms
- 按钮 active：`scale(0.97)`，160ms

### Typography System
- 标题：`DM Sans`（粗体，现代感）
- 正文：`Inter`（可读性强）
- 代码/Markdown：`JetBrains Mono`
- 层级：32px 页面标题 / 18px 卡片标题 / 14px 正文 / 12px 元数据

### Brand Essence
**SmartTodo**：专为个人打造的 AI 增强待办系统，让思维流动，让执行精准。
- 形容词：精准、智能、专注

### Brand Voice
- 标题风格：直接、有力，无废话
- 示例：「用自然语言写下任何事，AI 帮你整理好」
- CTA：「AI 整理」「立即添加」「查看今日」
- 禁用：「欢迎使用」「开始体验」

### Wordmark & Logo
蓝紫渐变的「✓ + 闪电」组合图标，代表完成 + AI 加速
